# Community

## Projects

### Home
This is the GitHub Page for this repository.
The stack is Vite + LitHTML + Haunted (it reads and writes
pretty similarly to React).

To start developing:
```sh
mise home-dev
```

### Feeds
Takes in a bunch of feeds and spits back out re-organized
feeds for the "Home" project.

To start developing, there is no hot-reload, so you'll have to
run
```sh
mise run feeds
```
to generate the latest feeds.
